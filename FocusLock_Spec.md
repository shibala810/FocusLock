# FocusLock(專注鎖)— 產品規格書

## 一、產品概述

一款 iOS app,幫高中生在準備大學考試期間自我約束手機使用。使用者(學生)可以排程或即時鎖定娛樂類 app,鎖定期間若想提前解鎖,必須答對指定科目的考題(高一至高二範圍),藉此製造解鎖摩擦並順便複習。

**核心定位:** 自我約束工具,不是家長監控工具。解鎖難度的目的是「讓衝動冷靜下來 + 順便讀書」,而非絕對鎖死。

---

## 二、技術前提(影響 UI 設計的關鍵限制)

1. **使用 Apple Screen Time API**(`FamilyControls` / `ManagedSettings` / `DeviceActivity`)。第三方 app 無法鎖住「整支手機」,只能封鎖使用者指定的 app(把它們變成無法開啟的灰色盾牌畫面)。所以 UI 上要讓使用者「選擇要封鎖哪些 app」。
2. 選 app 必須用 Apple 官方的 `FamilyActivityPicker`(系統元件,基於隱私,我們的 app 看不到使用者選了哪些 app 的名稱/圖示,只拿到一個加密 token)。**因此 UI 上無法顯示「已選 IG、TikTok」這種具體清單,只能顯示「已選擇 N 個 app/類別」。設計時要接受這個限制。**
3. 需要 Family Controls entitlement(向 Apple 申請)。授權採 `.individual` 模式(自己約束自己,不需家長 Apple ID)。
4. 題庫存在本地(JSON),離線可用,可持續擴充。

---

## 三、功能清單

### 1. 鎖定模式(兩種)
- **即時鎖定:** 選擇時長(快捷:30 分鐘 / 1 小時 / 2 小時 / 3 小時,或自訂),按下立即開始封鎖。
- **排程鎖定:** 設定每週重複的時段(例如 週一至週五 19:00–22:00),時間到自動封鎖、結束自動解除。可建立多組排程。

### 2. 封鎖對象設定
- 透過系統 `FamilyActivityPicker` 選擇要封鎖的 app 與類別。
- 一組設定可重複套用到即時鎖定與各排程。

### 3. 解鎖摩擦機制(核心特色)
鎖定期間若想「提前解鎖」,需通過以下流程(各項可在設定中開關/調整):
- **冷靜倒數:** 按下「我要解鎖」後,強制等待 N 分鐘(預設 5 分鐘)才能進入答題,中途離開重來。
- **答題解鎖:** 從題庫隨機抽 N 題(預設 3 題),全部答對才解鎖。答錯需重抽重答。
- **科目選擇:** 使用者可勾選本次解鎖要考哪些科目(數學 / 歷史 / 國文 / 英文),系統從勾選科目隨機出題。
- **每日解鎖次數上限:** 預設一天最多提前解鎖 3 次,用完當天不能再提前解鎖。
- **緊急解鎖:** 保留一個「緊急」按鈕(例如真的有急事),但會記錄次數並顯示在統計中,製造心理摩擦。

### 4. 題庫
本地 JSON,涵蓋四科,範圍高一至高二:
- 數學、歷史、國文、英文
- 每題包含:科目、年級、題幹、選項(四選一)、正解、(可選)解析
- 題型以單選題為主(方便在手機上快速作答與自動批改)
- 規格附帶每科範例題,使用者/開發者可持續擴充

### 5. 統計與激勵
- 今日/本週累積專注時間
- 提前解鎖次數、緊急解鎖次數
- 答題正確率(可視為「順便複習」的成果)
- 連續達標天數(streak)

### 6. 設定
- 解鎖題數、冷靜倒數時長、每日解鎖上限
- 預設出題科目
- 通知提醒(排程即將開始前提醒)

## 四、頁面與 UI 描述

### 頁面 0:歡迎 / 授權頁(首次啟動)
- App logo 與一句標語(例:「鎖住分心,解鎖靠實力」)
- 簡短說明三步驟:選 app → 設定鎖定 → 答題解鎖
- 主按鈕「開始設定」→ 觸發系統 Family Controls 授權彈窗
- 授權後進入主頁

### 頁面 1:主頁(Home / Dashboard)
頁面頂部:
- 大標題顯示目前狀態:**「未鎖定」** 或 **「鎖定中 ⏱ 剩餘 01:23:45」**(鎖定中時用醒目的色塊與倒數計時)
中段(未鎖定時顯示「即時鎖定」卡片):
- 時長快捷膠囊按鈕:30 分 / 1 時 / 2 時 / 3 時 / 自訂
- 大主按鈕「立即鎖定」
中段(鎖定中時):
- 倒數計時大字
- 次要按鈕「我要提前解鎖」(點下進入解鎖流程)
下段:
- 今日專注時間、streak 的小型統計卡
底部 Tab Bar:**首頁 / 排程 / 統計 / 設定**

### 頁面 2:封鎖對象設定頁
- 說明文字:基於 Apple 隱私規範,只顯示已選數量
- 卡片顯示「已封鎖 N 個 app・M 個類別」
- 按鈕「選擇要封鎖的 App」→ 開啟系統 FamilyActivityPicker
- 提示:此清單會套用到即時鎖定與所有排程

### 頁面 3:排程列表頁
- 列出已建立的排程,每筆顯示:名稱、重複的星期(圓點標示 一二三四五六日)、時段(19:00–22:00)、啟用開關 toggle
- 右上「+」新增排程
- 點一筆進入編輯

### 頁面 3b:排程編輯頁
- 排程名稱(例:晚自習)
- 開始時間 / 結束時間(時間滾輪)
- 重複星期(七個可點選的圓圈)
- 是否套用解鎖摩擦機制(開關)
- 儲存 / 刪除

### 頁面 4:解鎖流程(多步驟,全螢幕沉浸式)
**步驟 A — 確認頁**
- 文字提醒:「確定要提前解鎖嗎?今天還剩 X 次機會。」
- 兩個按鈕:「繼續(開始冷靜倒數)」/「算了,繼續專注」(返回)

**步驟 B — 冷靜倒數頁**
- 大型圓形倒數動畫,顯示剩餘秒數
- 文字:「深呼吸,N 分鐘後可以開始答題」
- 離開會中止並提示需重來

**步驟 C — 答題頁**
- 進度指示:第 1 / 3 題
- 顯示科目標籤(數學/歷史/國文/英文)與年級
- 題幹文字(數學題可能含公式,用清晰排版)
- 四個選項按鈕(A/B/C/D)
- 選答後即時回饋對/錯;答錯該題重抽
- 全部答對 → 步驟 D

**步驟 D — 解鎖成功頁**
- 正向回饋:「解鎖成功!順便複習了 3 題,正確率 100%」
- 顯示本次解鎖後 app 恢復可用
- 按鈕返回首頁

**緊急解鎖入口**(在步驟 A 角落,低調的文字連結)
- 點下需二次確認,並提示會被記錄

### 頁面 5:統計頁
- 本週專注時間長條圖(七天)
- 數字卡:總專注時數、提前解鎖次數、緊急解鎖次數、答題正確率
- streak 連續達標天數(火焰圖示)
- (可選)各科答題正確率,讓使用者看到自己哪科較弱

### 頁面 6:設定頁
分組列表:
- **解鎖難度**:解鎖題數(1–5)、冷靜倒數時長(0–15 分)、每日提前解鎖上限(0–10 次)
- **出題科目**:數學/歷史/國文/英文 的勾選開關
- **通知**:排程開始前提醒(開關 + 提前幾分鐘)
- **封鎖對象**:跳到頁面 2
- **題庫**:顯示目前各科題數(之後可做匯入)
- **關於**:版本、隱私說明

## 五、視覺風格建議(給 Claude Design 的方向)
- 氣質:專注、沉穩、帶一點鼓勵感,避免懲罰/壓迫感(對象是壓力大的考生)
- 主色建議:沉穩的深藍或墨綠當主色,搭配一個溫暖的強調色(琥珀/珊瑚)用在「鎖定中」「解鎖成功」等狀態
- 鎖定中狀態:用較深的背景 + 大倒數字,營造「進入專注結界」的儀式感
- 答題頁:乾淨、留白多、字大,像考卷但更友善
- 圓角卡片、柔和陰影、清楚的狀態色(專注=綠、解鎖中=琥珀、緊急=紅)
- 深色模式友善(晚自習時段常在暗環境使用)
